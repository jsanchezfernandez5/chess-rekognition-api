Parámetros
Detección del tablero
Umbral Hysteresis (Canny)0.33
Ajusta la sensibilidad del detector de bordes. Un valor menor (0.1) detecta bordes muy suaves; un valor mayor (0.5) solo detecta contrastes fuertes. Ideal: 0.25 - 0.40.
Filtro de Área Mínima8k
Descarte de ruido por tamaño (px²). Si el tablero está lejos, usa 4k-8k. Si está cerca, sube a 15k-30k para evitar que otros objetos interfieran.
Suavizado Temporal5
Número de frames para la media móvil de las esquinas. Elimina el parpadeo (jitter) y estabiliza la homografía. Recomendado: 4-6.
Límite de Aspecto (Ratio)2.2
Relación máxima entre ancho y alto del tablero proyectado. Si grabas con un ángulo muy rasante (perspectiva extrema), aumenta este valor a 3.0+.
Detección de piezas
Sensibilidad de Ocupación35
Umbral de desviación estándar (σ) cromática. Determina si el contenido de una casilla difiere lo suficiente del fondo para considerarse una pieza.
Procesamiento de imagen
Realce CLAHE Ecualización adaptativa del contraste por bloques. Crucial para mitigar sombras y reflejos en el tablero.
Filtrado Geométrico Asegura que el polígono sea convexo y cumpla simetría básica de un tablero. Evita falsas detecciones en fondos complejos.
Máscara ROI Dibuja la cuadrícula de enfoque para ayudar al usuario a encuadrar el tablero centralmente.
